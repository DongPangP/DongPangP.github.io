如何使用jQuery？

下载jQuery库

下载地址：http://code.jquery.com/

引入下载的jQuery库

<script src="code/js/jquery-1.12.4.js"></script>

编写jQuery代码

// 原生JS的固定写法
window.onload = function(ev){}

// jQuery的固定写法
$(document).ready(function(){
   ...
});

jQuery版本区别

1.x ：兼容ie678，但相对其他版本文件较大，官方只做BUG维护，功能不再新增，最终版本：1.12.4（2016年5月20日）

2.x：不兼容ie678，相对1.x文件较小，官方只做BUG维护，功能不再新增，最终版本：2.2.4（2016年5月20日）

3.x：不兼容ie678，只支持最新的浏览器，很多老的jQuery插件不支持这个版本，相对1.x文件较小，提供不包含ajax/动画API版本

jQuery入口函数

jQuery与原生js加载模式的区别

加载内容不同

​ 原生js会在dom原生与图片内容都加载后执行；

​ jQuery会在dom元素加载后执行，不会等待图片加载。

多个入口文件时执行方式不同

​ 原生js只执行最后一个入口文件的内容，即后面的会覆盖前面的；

​ jQuery多个都执行，前面的执行完才执行后面的

jQuery入口函数的写法

$(document).ready(function(){
    ...
});

jQuery(document).ready(function(){
    ...
});

$(function(){
 ...
});

jQuery冲突问题

​ 当引入多个js文件后，发生 $ 符号重复冲突时

​ 解决方法1：释放 $ 的使用权

​ 注意点：释放操作必须在编写其他 jQuery 代码之前；释放之后不能再使用 $，改为使用 jQuery

jQuery.noConflict();  //释放操作
jQuery(function(){
    alert('hello world');
});

​ 解决方法2：自定义一个访问符号

var mmp = jQuery.noConflict();
mmp(function(){
    alert('hello mmp');
});

jQuery核心函数

$(); 就代表调用 jQuery 的核心函数

// 1、接收一个函数
$(function (){
  //2、接收一个字符串
  //2.1、接收一个字符串选择器,返回一个jQuery对象，对象中保存了找到的DOM对象
  var $box1 = $(".box1");
  var $box2 = $("#box2");
  //2.2、接收一个字符串代码片段,返回一个jQuery对象,对象保存了DOM元素
  var $p = $("<p>我是段落</p>");
  $box1.append($p);
  // 3、接收一个DOM元素，返回一个jQuery对象
  var span = document.getElementByTagName("span")[0];
  var $span = $(span);
  console.log($span);
});

jQuery 静态方法

each

用于遍历指定的对象和数组，即可以遍历伪数组

each方法 遍历谁就返回谁

$.each( object,callback );
// 举例 index 索引 value 值
$(function(){
    $.each([23,32],function(index,value){
        alert(index+':'+value);
    })
});

map

描述：将一个数组中的所有元素转换到另一个数组中

jQuery.map(array,callback(elementOfArray,indexInArray))

array ：数组

callback(elementOfArray,indexInArray)：处理每一个元素的函数。第一个参数是数组元素，第二个参数是该元素的索引值。该函数可以返回任何值，在函数内部，this将是全局window对象

jQuery.map(arrayOrObject,callback(value,indexOrKey))

arrayOrObject：待转换数组或对象

callback(value,indexOrKey)：处理每一个元素的函数。第一个参数是数组中或对象的值，第二个参数就是该元素在数组中的索引值或该对象的键。该函数可以返回任何值，该返回值会被添加到数组中。若返回时数组，则会将该数组中的元素添加到最终的结果数组中，在函数内部，this将是全局window对象

在jQuery 1.6之前，$.map()只支持遍历数组和类似数组的对象 。在jQuery 1.6也支持遍历对象。

trim

去除字符串前后空格

var str = "  ls  ";
var res = $.trim(str);
console.log("---"+str+"---");
console.log("---"+res+"---");

isWindow

判断传入的对象是否是window对象

返回值：true/false

isArray

判断传入的对象是否为数组

返回值：true/false

isFunction

判断传入的对象是否为函数

返回值：true/false

:empty

作用：找到既没有文本内容也没有子元素的指定元素

var $div = $("div:empty");

:parent

作用：找到有文本内容或有子元素的指定元素

var $div = $("div:parent");

:contains

作用：找到包含指定文本内容的指定元素

var $div = $("div:contains('我是div')");

:has

作用：找到包含指定子元素的指定元素

var $div = $("div:has('span')");

属性和属性节点

attr(name|pro|key,val|fn)

获取或者设置属性节点的值

如果传递一个参数，代表获取属性节点的值

如果传递两个参数，代表设置属性节点的值

注意：

  • 获取，无论找到多少个元素，都只会返回第一个元素指定的属性节点的值
  • 设置，找到多少个元素就会设置多少个元素
  • 设置，如果设置的属性节点不存在，那么系统会自动新增
console.log($("span").attr("class"));
$("span").attr("class","box");
$("span").attr("abc","123");

removeAttr(name)

删除属性节点

注意：会删除所有找到元素指定的属性节点

// 删除所有span中包含的 class
$("span").removeAttr("class");

prop

removeProp

使用

官方推荐在操作属性节点时，具有 true 和 false 两个属性的属性节点，如checked、selected或者disabled 使用prop()，其他的使用 attr()

$(function(){
    var btn = document.getElementsByTagName("button")[0];
    btn.onclick = function(){
        var input = document.getElementsByTagName("input")[0];
        var text = input.value;
        $("img").attr("src",text);
        //$("img").prop("src",text); //效果一样
    }
})

JQuery操作CSS类的相关方法

addClass(class|fn)

添加类，如果要添加多个，多个类名之间用空格隔开即可。

$(function(){
    var btns = document.getElementsByTagName("button");
    btns[0].onclick = function(){
        $("div").addClass("class1 class2");
    }
})

removeClass

删除类，如果要删除多个，多个类名之间用空格隔开即可

toggleClass

切换类，有就删除，没有就添加
