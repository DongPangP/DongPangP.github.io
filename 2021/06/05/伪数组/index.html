伪数组

2021-06-05JavaScript高级程序设计笔记 • 对象

伪数组的特点

  • Object 对象
  • length 属性
  • 数值下标属性
  • 没有数组特别的方法：forEach() ，push()，pop()，splice()
const obj = {
      0: tom,
      1: jerry,
      2: bob
      length: 3
}

如何判断一个对象是否属于伪数组

Array.isArray

console.log(Array.isArray(obj)) // false;

instanceof

console.log(obj instanceof Array) //false

常见的伪数组

  1. jQuery 对象（比如 $(“div”)）
  2. 函数内部的arguments
  3. DOM中 NodeList对象

将伪数组转换成数组

可以使用Array.prototype.slice.call() 将数组转化为真正的Array对象

const arr = Array.prototype.slice.call(obj);
